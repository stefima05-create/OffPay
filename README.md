## Problem
Digital payments fail in areas with poor or no internet connectivity, especially in rural regions, markets, and during network outages.

## Solution
OffPay is an offline-first digital wallet that allows peer-to-peer payments using Bluetooth or local connections. Transactions are stored locally and synced once internet connectivity is restored.

## Key Features
- Offline peer-to-peer payments
- Local transaction storage
- Auto-sync when internet returns
- Low-end device support

## Tech Stack
- Android (Kotlin)
- Jetpack Compose
- Local storage for offline ledger

## Hackathon Goal
To enable reliable digital payments even without internet connectivity.
